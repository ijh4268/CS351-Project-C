CS351 - Project C

Instructions:

  1. Use the 'w', 'a', 's', and 'd' keys to move around the scene.
  2. Use the arrow keys to adjust the viewing angle of the camera.
  3. You may select which Material you'd like to use using the dropdown below.
  4. You may input different 'R' (red), 'G' (green), 'B' (blue), value to change the color of the ambient, diffuse, and specular light.
  5. You can also adjust the 'X', 'Y', and 'Z' coordinates of the light source using the input boxes.
  6. You can toggle back and forth between Phong Shading and Gouraud Shading, as well as between Phong lighting and Blinn-Phong lighting.

Please use a browser that supports "canvas"
